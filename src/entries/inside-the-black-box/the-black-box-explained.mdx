## Understanding a Neural Network
Let's start with understanding the neural network. Not as a set of digital neurons
composed in layers but as a function approximator. The goal of a neural network is to produce
the correct result for the given input arguments. The arguments might be whatever they are 
(in a form of numbers tho) - set of pixels data, paragraph of words, financial report numbers, etc.
The task for the neural network, and I'm talking about neural networks in general,
is to produce the correct response for the input data. And the question is what does it mean?
